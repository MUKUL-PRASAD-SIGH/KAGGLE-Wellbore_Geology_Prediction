# ROGII - Wellbore Geology Prediction

## Overview

This repository contains my solution for the Kaggle competition:

ROGII – Wellbore Geology Prediction

The objective is to predict geological formations encountered along horizontal wellbores using drilling and subsurface data.

Competition Link:
https://www.kaggle.com/competitions/rogii-wellbore-geology-prediction

---

## Problem Statement

Develop machine learning models capable of predicting geological properties along a horizontal well path to improve drilling efficiency, reduce operational risks, and support automation in the oil & gas industry.
